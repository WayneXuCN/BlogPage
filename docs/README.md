# 博客项目文档

欢迎使用这个基于 Astro 的博客系统！本项目是一个现代化、响应式、可访问且对 SEO 友好的博客主题，名为 AstroPaper。

## 目录结构

- [配置指南](./configuration.md) - 如何配置博客网站
- [功能使用](./features.md) - 如何使用博客的各项功能
- [撰写与发布](./writing-posts.md) - 如何撰写和发布博客文章
- [多语言支持](./i18n.md) - 如何使用和配置多语言功能
- [主题定制](./customization.md) - 如何定制博客主题和样式

## 项目特性

- 类型安全的 Markdown 内容
- 高性能表现
- 无障碍访问（键盘/语音浏览）
- 响应式设计（移动设备到桌面）
- SEO 友好
- 亮色/暗色模式
- 模糊搜索功能
- 草稿文章和分页
- 站点地图和 RSS 订阅
- 遵循最佳实践
- 高度可定制
- 动态 OG 图片生成

## 快速开始

要开始使用此博客系统，请按照以下步骤操作：

1. 安装依赖：`bun install`
2. 启动开发服务器：`bun run dev`
3. 访问 `http://localhost:4321`

更多信息请参阅各个文档页面。
