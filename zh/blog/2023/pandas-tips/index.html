Python 数据处理技巧：Pandas 高级用法

分享一些在日常数据分析中常用的 Pandas 高级技巧，包括数据清洗、聚合操作和性能优化。

徐文杰 2023年3月10日 更新于 2023年11月25日 1 分钟
#Python #Pandas #数据分析 #教程

引言

Pandas 是 Python 数据分析的核心库，本文分享一些实用的高级技巧。

本文示例代码基于 Pandas 2.0+ 版本，部分新特性可能在旧版本中不可用。

1. 链式调用与 pipe()

使用 pipe() 方法可以让数据处理流程更加清晰：

import pandas as pd

def clean_data(df):
    return df.dropna().reset_index(drop=True)

def add_features(df):
    df['year'] = pd.to_datetime(df['date']).dt.year
    return df

# 链式调用
result = (
    df
    .pipe(clean_data)
    .pipe(add_features)
    .query('year >= 2020')
)

2. 高效的数据聚合

使用 agg() 方法进行多维度聚合：

# 多列多函数聚合
summary = df.groupby('category').agg({
    'value': ['mean', 'std', 'count'],
    'score': ['min', 'max']
})

# 自定义聚合函数
def range_func(x):
    return x.max() - x.min()

df.groupby('category')['value'].agg(['mean', range_func])

3. 内存优化

处理大数据集时，类型优化很重要：

# 优化整数类型
def optimize_int(df, col):
    c_min = df[col].min()
    c_max = df[col].max()
    
    if c_min >= 0:
        if c_max < 255:
            return df[col].astype('uint8')
        elif c_max < 65535:
            return df[col].astype('uint16')
    
    if c_min > -128 and c_max < 127:
        return df[col].astype('int8')
    elif c_min > -32768 and c_max < 32767:
        return df[col].astype('int16')
    
    return df[col]

4. 向量化字符串操作

避免使用 apply()，优先使用向量化方法：

# ❌ 慢
df['name'].apply(lambda x: x.lower().strip())

# ✅ 快
df['name'].str.lower().str.strip()

# 正则表达式提取
df['phone'].str.extract(r'(\d{3})-(\d{4})-(\d{4})')

5. 条件赋值

使用 np.where() 或 np.select() 进行条件赋值：

import numpy as np

# 简单条件
df['status'] = np.where(df['score'] >= 60, 'pass', 'fail')

# 多条件
conditions = [
    df['score'] >= 90,
    df['score'] >= 80,
    df['score'] >= 60,
]
choices = ['A', 'B', 'C']
df['grade'] = np.select(conditions, choices, default='D')

总结

掌握这些技巧可以显著提升数据处理效率。记住：

  1. 优先使用向量化操作
  2. 合理使用数据类型
  3. 善用链式调用保持代码清晰

Happy coding! 🐍
