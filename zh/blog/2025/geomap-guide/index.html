学习如何使用 GeoMap 组件在博客中展示交互式地图

2025年1月17日 1 分钟

本文介绍如何使用 GeoMap 组件在博客中展示地图。

关于 Leaflet

GeoMap 组件基于 Leaflet 库，支持标记点、GeoJSON 数据展示等功能。

使用方式

import { GeoMap } from '@/components/mdx';

<GeoMap
  center={[纬度, 经度]}
  zoom={缩放级别}
  markers={[
    { lat: 纬度, lng: 经度, popup: "标记说明" }
  ]}
  height="400px"
/>
