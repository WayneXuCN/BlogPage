本文介绍如何在博客中使用 Typograms 组件创建 ASCII 艺术图表。

<Callout type="info" title="关于 Typograms">
Typograms 将纯文本 ASCII 图形渲染为高质量的 SVG，非常适合创建简单的流程图和示意图。
</Callout>

## 使用方式

```jsx
import { Typograms } from '@/components/mdx';

<Typograms>
{`
+--------+     +--------+
| 开始   | --> | 结束   |
+--------+     +--------+
`}
</Typograms>
```

## 绘图语法

- `+` 角落连接点
- `-` 水平线
- `|` 垂直线
- `>`, `<`, `^`, `v` 箭头方向
- 文字直接写在框内
