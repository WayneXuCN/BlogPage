## 使用方式

```jsx
import { PhotoGallery } from '@/components/mdx';

<PhotoGallery
  images={[
    {
      src: "完整图片URL",
      thumb: "缩略图URL",
      alt: "图片描述",
      title: "图片标题"
    },
    // 更多图片...
  ]}
  columns={3}
/>
```

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| images | ImageItem[] | 必填 | 图片数组 |
| columns | 2 \| 3 \| 4 | 3 | 每行显示的列数 |
