import { GeoMap, Callout } from '@/components/mdx';

本文介绍如何使用 GeoMap 组件在博客中展示地图。

<Callout type="info" title="关于 Leaflet">
GeoMap 组件基于 Leaflet 库，支持标记点、GeoJSON 数据展示等功能。
</Callout>

## 基础地图

展示一个简单的地图，标记北京的位置：

<GeoMap
  center={[39.9042, 116.4074]}
  zoom={11}
  markers={[
    { lat: 39.9042, lng: 116.4074, popup: "北京市中心" },
    { lat: 39.9087, lng: 116.3975, popup: "故宫博物院" },
    { lat: 40.0020, lng: 116.3958, popup: "鸟巢" }
  ]}
  height="400px"
/>

## 使用方式

```jsx
import { GeoMap } from '@/components/mdx';

<GeoMap
  center={[纬度, 经度]}
  zoom={缩放级别}
  markers={[
    { lat: 纬度, lng: 经度, popup: "标记说明" }
  ]}
  height="400px"
/>
```
