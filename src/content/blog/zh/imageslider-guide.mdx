## 使用方式

```jsx
import { ImageSlider } from '@/components/mdx';

<ImageSlider
  images={[
    {
      src: "图片URL",
      alt: "图片描述",
      caption: "图片标题"
    },
    // 更多图片...
  ]}
  autoplay={true}
  showPagination={true}
  showNavigation={true}
/>
```

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| images | ImageItem[] | 必填 | 图片数组 |
| autoplay | boolean | false | 是否自动播放 |
| showPagination | boolean | true | 是否显示分页指示器 |
| showNavigation | boolean | true | 是否显示导航箭头 |
