## System Configuration

### 1. Zsh Installation and Configuration

#### Install Zsh

```sh
# Install Zsh
sudo apt update
sudo apt install zsh

# Check Zsh version
zsh --version

# Set Zsh as default shell
chsh -s $(which zsh)
```

#### Install Zimfw

> 2025.02.16: Using Zimfw instead of Oh My Zsh for better integration and efficiency

```sh
# Check current shell
echo $SHELL
# Install Zimfw via curl
curl -fsSL https://raw.githubusercontent.com/zimfw/install/master/install.zsh | zsh
echo $ZSH

# Backup .zshrc configuration file
cp ~/.zshrc ~/.zshrc.bak
# Remove Oh My Zsh configuration
rm -rf ~/.oh-my-zsh

# Edit zsh configuration file
nano ~/.zshrc
# Edit zim configuration file
nano ~/.zimrc
# Reload configuration via source
source ~/.zshrc
```

#### Zsh Plugins and Theme Configuration

```sh
# Add powerlevel10k module
zmodule romkatv/powerlevel10k
zimfw install
zimfw update
zimfw upgrade

# Install Zoxide (not via Zimfw)
curl -sSfL https://raw.githubusercontent.com/ajeetdsouza/zoxide/main/install.sh | sh
```

> Most plugins that require manual configuration in Oh My Zsh (such as compinit, SHARE_HISTORY, CORRECT, etc.) are enabled by default in Zim. You only need to manually add the required modules and themes, then use `zimfw install` to automatically install them.
